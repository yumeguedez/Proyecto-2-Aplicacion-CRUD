Administrador de Pagos Casero

¿Tienes muchas deudas y no sabes como realizar la administración de las mismas?

Con el administrador de pagos puedes crear tu deuda, almacenarla, editarla (si aplica) y eliminarla una vez hayas realizado el pago

Solo debes seguir las siguientes instrucciones:

  • Crea una deuda con el nombre y el monto y presiona el boton CREAR
  • Si deseas editar el monto o nombre de la deuda presiona el boton EDITAR
  • Una vez hayas realizado el pago presiona el boton PAGADO
  • Si ya la deuda no aplica presiona el boton ELIMINAR
